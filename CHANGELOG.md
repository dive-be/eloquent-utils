# Changelog

All notable changes to `eloquent-utils` will be documented in this file.

## 0.3.0 - 2024-03-14

### Added

- PHP 8.3 support
- Laravel 11 support

### Removed

- PHP 8.2 support
- Laravel 10 support

## 0.2.0 - 2023-04-25

### Added

- PHP 8.2 support
- Laravel 10 support

### Removed

- PHP 8.1 support
- Laravel 9 support

## 0.1.2 - 2023-04-03

### Fixed

- `$model->is(...)` equality checks will now succeed when compared with models retrieved from the database.

## 0.1.1 - 2022-05-24

### Added

- `InteractsWithStaticData` Use in-memory cache to resolve route binding.

## 0.1.0 - 2022-05-23

- Initial release
